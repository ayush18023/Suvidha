# TRINIT_HAX_DEV01



Steps to Run:
```
git clone https://github.com/ayush18023/TRINIT_HAX_DEV01.git
cd TRINIT_HAX_DEV01
```
Frontend
```
cd frontend 
npm i
npm start
```

Backend
```
cd backend
npm i 
nodemon server.js
```
